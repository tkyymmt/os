## Features
- ugly original bootloader
- original file system
- paging
- non preemptive single tasking
- flat binary loading

## Document
<https://github.com/takuya-devel/os/blob/master/document.txt>

## Requirements
- binutils
- gcc
- qemu
- make

## Build
In root of the repository:
```
$ make run
```
